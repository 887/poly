# poly-demo

Demo/mock messenger client for **Poly** (PolyGlot Messenger).

## Purpose

Implements the `ClientBackend` trait with randomly generated fake data. Used for UI development and testing without requiring real messenger accounts.

## Features

- Generates demo servers, categories, channels (text/voice/video)
- Generates demo users with avatars and presence
- Generates demo messages with varied content types
- Generates demo friend lists, group chats, notifications
- Simulates real-time events (new messages, presence changes, typing)

## Usage

Enable in Poly settings as a "Demo Account" backend. Instantly populates the entire UI with realistic mock data.
