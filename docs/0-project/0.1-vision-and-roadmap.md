## Vision

**Poly** (PolyGlot Messenger) is a cross-platform multi-backend messenger aggregator built with Rust + Dioxus 0.7.3. It unifies Discord, Matrix, Stoat (Revolt), Microsoft Teams, and custom servers into one Discord-style UI.

### Core Value Proposition

- **One client, many networks** — chat across all backends without switching apps
- **Multi-account** — multiple accounts per backend (3 Discord + 2 Matrix, etc.)
- **Favorites-driven UX** — curate a personal server sidebar from across all backends
- **Cross-platform** — Desktop (Win/Mac/Linux), Mobile (Android/iOS), Web — single codebase
- **Encrypted cloud backup** — settings synced to user-controlled servers with E2E encryption
- **Privacy-first identity** — Ed25519 keypair identity (Session Messenger model), no email/phone
- **WASM plugin system** — backends are sandboxed WASM Component Model binaries

### Design Philosophy

- Look and feel like Discord/Stoat — familiar enough to pull Discord users in
- Act like Thunderbird — aggregate backends as Thunderbird aggregates email providers
- Polyglot client, not a feature-complete replacement for any single platform's official client
- Feature-flag each backend: builds with only the platforms needed

### Phase 2 — Core Infrastructure (DONE)
- Monorepo layout, shared library crate (`poly-core`)
