## Cryptography

- `ed25519-dalek` — Ed25519 identity key generation (Session Messenger model)
- `x25519-dalek` — X25519 ECDH key exchange
- `bip39` — mnemonic phrase for key recovery
- `chacha20poly1305` / `aes-gcm` — symmetric encryption for backup
