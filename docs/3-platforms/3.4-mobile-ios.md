## Status

Not yet started. Dioxus 0.7.3 supports iOS (iPad support added in 0.7.x), but mobile-specific work is deferred post-Phase 3.

## Key Considerations

### WASM Plugins on iOS

JIT is prohibited by iOS. WASM plugins require AOT compilation:
- `engine.precompile_component()` → `.cwasm` file
- Pre-compile during build step OR on first launch, cache to app sandbox
- Alternatively: ship pre-compiled `.cwasm` files per architecture (arm64)

### Dioxus iOS

- `dx serve` with ADB reverse proxy equivalent for iOS
- `Dioxus.toml` — customizable `Info.plist`
- Hot-reload support via USB bridge

### UI Considerations

Three-pane layout collapses to swipeable pages on mobile:
- Swipe left → server / channel list
- Center → active chat
- Swipe right → member list
