## Status

Not yet started. Dioxus 0.7.3 supports Android, but mobile-specific work is deferred post-Phase 3.

## Key Considerations

### WASM Plugins on Android

wasmtime's Cranelift supports aarch64. JIT is allowed on Android:
- First load: compile `.wasm` → native, cache result
- Subsequent loads: use cached precompiled binary

### Dioxus Android

- `Dioxus.toml` — customizable Android manifest
- ADB reverse proxy for dev server access during development
- `dx serve` with `--platform android` (less tested than web/desktop)

### Build Output

- APK via `dx build --platform android`
- AAB for Play Store distribution

### UI Considerations

Same three-pane layout as iOS:
- Swipe left → server / channel list
- Center → active chat
- Swipe right → member list
