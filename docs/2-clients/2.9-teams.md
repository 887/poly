## Overview

Microsoft Teams backend via the Microsoft Graph API.

**Crate:** `clients/teams/` (`poly-teams`) — stub
**Feature flag:** `teams` / `dev-plugins`

## API Approach

Microsoft Graph API (`https://graph.microsoft.com/v1.0/`):
- OAuth2 / Azure AD authentication (MSAL flow)
- `/me/joinedTeams` — list Teams the user is in
- `/teams/{id}/channels` — channel list
- `/teams/{id}/channels/{id}/messages` — message history
- `/me/chats` — individual chats and group chats
- `/me/chats/{id}/messages` — chat message history
- Change notifications (subscription + webhook/polling) for real-time events
