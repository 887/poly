## Overview

Matrix is an open federated messaging protocol. The Poly client wraps `matrix-sdk = "0.16.0"` — the official Rust SDK that powers Element X.

**Crate:** `clients/matrix/` (`poly-matrix`)
**WASM plugin:** planned (`wasm32-wasip2`)

## Status

**Stub** — crate exists, `matrix-sdk` dependency is present, but implementation is minimal. Core chat flow not yet functional.

## matrix-sdk

- Version: `0.16.0` (same SDK used by Element X iOS/Android)
- Handles: login, sync, message send/receive, E2EE
- Authentication: password login or SSO / token

## Poly Mapping

| Matrix concept | Poly concept |
|---|---|
| Homeserver | Account instance (e.g. `matrix.org`) |
| Room (no Space) | Fake server category "Direct" or "Rooms" |
| Room (in Space) | Channel under a Space (server) |
| Space | Server |
| DM room | DM channel |

Matrix doesn't have Discord-style servers natively, but Spaces provide the grouping. Rooms without a Space are grouped under a synthetic server per account.
