## Design Principles

- Each backend gets its own standalone axum binary
- Dynamic port via `TcpListener::bind("127.0.0.1:0")`
- In-memory state (`DashMap` / `RwLock<HashMap>`) â€” no database overhead
- `/reset` = `.clear()`, fully deterministic
- `POST /test/auth/token` â€” no-password test login for all backends
