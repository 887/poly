# 6.3 Memory MCP — IN-PROGRESS

> Status: IN-PROGRESS
> Sources: docs/archive/phases/phase-5.2-plan.md
> Last updated: 2026-04-12

## Overview

`poly-memory` MCP server — provides persistent memory, task tracking, and knowledge storage for AI agents working on the Poly project. Separate from `poly-chat-mcp`.

**Crate:** `mcp/` (loaded as `poly-memory` in Claude Code sessions)

## Tools

| Tool | Description |
|---|---|
| `store_memory` / `load_memories` | Persistent agent memory |
| `store_knowledge` / `get_knowledge` / `search_knowledge` | Project knowledge base |
| `store_finding` / `load_findings` | Findings from investigations |
| `create_task` / `get_task` / `list_tasks` | Task management |
| `add_task_item` / `check_task_item` | Checklist items |
| `set_task_status` / `next_task` / `redo_task` | Task lifecycle |
| `work_plan` | Get current work plan |
| `task_start_reminders` | Reminders shown on task start |

## Relationship to Social Agent (Phase 5.2)

The social agent's "per-chat memory" and "personality storage" are separate from this project-meta memory MCP. The memory MCP is for AI coding agents; the social agent memory is for per-contact conversation history and learned preferences.

Phase 5.2 will design the social memory system separately.
